# Changelog

All notable changes to TechiesLog will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-08

### Added
- Initial release of TechiesLog
- Dashboard with productivity overview
- Task management with priorities and due dates
- Todo list for quick capture
- Project tracking with progress monitoring
- Goal setting and tracking
- Habit tracking with streaks
- Daily logs and reflections
- Finance tracking (income/expense)
- Budget management
- Notes with tags and linking
- Dark/Light theme support
- Offline-first architecture
- SQLite database for local storage
- Privacy-focused (no ads, no analytics, no tracking)

### Features
- **Work Management**: Tasks and Todos with categories and priorities
- **Premium Hub**: Habits, Goals, and Projects tracking
- **Finance**: Income/Expense tracking with categories and budgets
- **Journal**: Daily logs and notes with mood tracking
- **Settings**: Theme customization and app preferences

### Technical
- Built with Flutter
- Package name: com.vinoth.techieslog
- Minimum SDK: Android 5.0 (API 21)
- Target SDK: Android 14 (API 34)
- 100% offline functionality
- No internet permissions required
